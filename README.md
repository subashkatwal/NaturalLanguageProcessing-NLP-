# Natural Language Processing Experiments

This repository contains early experiments and practice notebooks related to 
Natural Language Processing (NLP), document information extraction, and neural networks.

These notebooks were created as part of my learning process while exploring 
machine learning and NLP concepts.

## Contents

### 1. RNN.ipynb
This notebook demonstrates a basic implementation of a Recurrent Neural Network (RNN).

Topics covered:
- Data scaling using MinMaxScaler
- Preparing sequential data
- Building an RNN using Keras
- Training a simple neural network model
- Understanding sequence-based prediction

Libraries used:
- NumPy
- Scikit-learn
- Keras

---

### 2. DocumentExtraction.ipynb
This notebook explores extracting useful information from document images.

Features:
- OCR using Tesseract
- Image processing with PIL
- Text extraction from uploaded files
- Regex-based information extraction
- Basic Named Entity Recognition using spaCy

Example extracted fields:
- Invoice number
- Total bill amount
- User name
- Company name
- Location

Libraries used:
- pytesseract
- PIL
- spaCy
- Regex
- Python IO utilities
